Crack The Lock

Java, Android Studio, Xml (for layout)

This is my first android puzzle game, which is still in a process.
The game revolves around the concept of deducing the correct 4-digit combination by analyzing the number of guesses with accurate positions and correct digits.
The game also has different languages, which are based on each phone's system language.
After the game is finished, here will be a link to download it from Google Play :).
